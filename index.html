Introduction

Spells Protocol is an experimental "AI-to-Blockchain" pipeline. It allows users to prompt an LLM to generate functional HTML5 applications, which are then compressed and stored immutably on the Monad Blockchain.

Protocol Architecture

[ USER PROMPT ] --> [ GOOGLE GEMINI ] --> [ HTML CODE ]
                                     |
                                   [ LZ-STRING COMPRESSION ]
                                     |
[ MONAD BLOCKCHAIN ] <-- [ SSTORE2 POINTER ]

SSTORE2 Storage

Traditional NFTs store metadata on IPFS, which is mutable and can go offline. Spells Protocol uses the `SSTORE2` pattern to write bytecode directly to a contract address. This data is part of the blockchain state and cannot be deleted.

Runtime Sandbox

When launching an app, the browser performs a real-time decompression. To allow the app to interact with Web3 without compromising the user's security, we inject a secure bridge.

// The Injection Bridge (Simplified)
window.ethereum = window.parent.ethereum;
window.ethers = window.parent.ethers;
